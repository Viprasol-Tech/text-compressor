# Text Compressor

Compress text while preserving meaning

## Features

- Zero external dependencies (stdlib only)
- Easy-to-use CLI interface
- Professional Python implementation
- MIT licensed

## Installation

```bash
pip install -e .
```

Or clone and install:

```bash
git clone https://github.com/Viprasol-Tech/text-compressor
cd text-compressor
pip install -e .
```

## Usage

### Python

```python
from text_compressor import TextCompressor

result = TextCompressor.process("data")
print(result)
```

### CLI

```bash
python -m text_compressor "your input here"
```
